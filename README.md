**Student Database Management System (SQLite)**



A simple SQL-based database system to manage student records using SQLite.



* Features

1. Create student table
2. Insert new student records
3. Update student details
4. Delete student records
5. Search and filter students



* Concepts Used

1. SQL CRUD operations (Create, Read, Update, Delete)
2. Primary Key and Unique constraints
3. Basic relational database design
4. Query-based data retrieval



* Tools

1. SQLite
